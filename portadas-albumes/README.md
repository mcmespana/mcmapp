# Portadas de álbumes

Imágenes originales de las portadas de los álbumes de fotos de la app
(Pascuas, congresos, JMJ, Jubileo…). No se empaquetan en la app: las portadas
que ve el usuario se sirven por URL desde el nodo `albums` de Firebase. Esta
carpeta es el archivo fuente para subirlas/regenerarlas.
