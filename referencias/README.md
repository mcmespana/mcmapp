# Referencias

Material histórico y de referencia. Nada de esta carpeta se usa en el código de
la app — es documentación visual y ejemplos.

## `capturas/`

Capturas de pantalla de la app (portada, notificaciones, horario, materiales,
visitas, profundiza). Sirven como referencia visual de cómo se ven las
secciones de un evento maquetadas con los JSON de
`mcm-app/firebase-seed/eventos/`.

## `cantoral-antiguo/`

Canciones de ejemplo en formato ChordPro (`.cho`) de la versión antigua de la
app, cuando el cantoral vivía en el bundle. Hoy las canciones viven en Firebase
(`songs/data`), pero estos archivos sirven como ejemplo del formato ChordPro
usado (directivas `{title:}`, `{comment:}`, acordes `[C]`, etc.).
